# dwny

A simple CLI tool to download files from the internet. Supports resuming interrupted downloads.

## Installation

```bash
go install .
```

## Usage

```bash
dwny -u <url> [-o output]
```

## Features

- [x] Resume interrupted downloads
- [x] Show progress
- [ ] Download multiple files
- [ ] Show download speed
- [ ] Show download time remaining
- [ ] and more...
